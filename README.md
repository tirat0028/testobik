# Движение денежных средств (ДДС)

Веб-приложение для учета движения денежных средств.

## Технологии

- Python 3.10+
- Django 5.2+
- SQLite (встроенная в Django)
- Bootstrap 5.3

## Установка и запуск

1. Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv venv
   # Windows
   venv\Scripts\activate
   # Linux/macOS
   source venv/bin/activate
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Примените миграции (если не применялись ранее):
   ```bash
   python manage.py migrate
   ```

4. (Опционально) Добавьте начальные данные:
   ```bash
   python init_data.py
   ```

5. (Опционально) Создайте суперпользователя для доступа к админ-панели:
   ```bash
   python create_superuser.py
   ```
   Или вручную:
   ```bash
   python manage.py createsuperuser
   ```

6. Запустите сервер:
   ```bash
   python manage.py runserver
   ```

7. Откройте в браузере http://127.0.0.1:8000/

## Функциональность

- Просмотр списка транзакций с фильтрацией
- Создание, редактирование и удаление транзакций
- Управление справочниками (статусы, типы, категории, подкатегории) через админ-панель
- Автоматическое заполнение даты создания транзакции
- Зависимые списки: подкатегории от категории, категории от типа транзакции

## Доступ к админ-панели

Админ-панель доступна по адресу http://127.0.0.1:8000/admin/

Используйте данные суперпользователя для входа.
